## Sobre a landing page

A landing foi construída para:

- Apresentar minha oferta como **especialista em IA aplicada e automação de operações**.
- Destacar **resultados mensuráveis** (precisão em RAG, volume processado, crescimento de base etc.).
- Organizar minha experiência em um formato voltado a negócios, não apenas currículo tradicional.
- Facilitar contato direto via **WhatsApp**, **GitHub** e **LinkedIn**.

### Seções principais

- **Hero** – Nome, posicionamento, CTA para consultoria e links para GitHub/LinkedIn.
- **Métricas-chave** – Precisão de modelos, volume processado e crescimento de base.
- **Projetos & Soluções** – RAG jurídico, Sherlock Stock SaaS, app Foco e SDR Agents (WhatsApp).
- **Histórico Operacional** – Linha do tempo com as principais experiências em Data, AI e operações.
- **Contato** – CTA direto para WhatsApp e informações de contato.

## Stack utilizada

A página é um único arquivo `index.html`, usando:

- **HTML5** puro
- **Tailwind CSS via CDN**
- **Google Fonts (Inter)**
- Layout responsivo em **dark mode** com foco em legibilidade e conversão.

Não há build step nem dependências adicionais: basta abrir o `index.html` no navegador ou servir via GitHub Pages.

---

## Como rodar localmente

1. Clone este repositório:

   ```bash
   git clone https://github.com/codetradeai/codetradeai.github.io.git
   cd codetradeai.github.io
   ```

2. Abra o arquivo `index.html` diretamente no navegador  
   *(clique duas vezes no arquivo ou use uma extensão de live server se preferir).*

---

## Deploy

O deploy é feito automaticamente pelo **GitHub Pages** a partir da branch `main`.

- Tipo: **User site** (`codetradeai.github.io`)
- URL: `https://codetradeai.github.io/`

Qualquer alteração em `index.html` na branch `main` dispara uma nova publicação.
